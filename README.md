# AI Hand Gesture Calculator

AI based hand gesture calculator using Python, OpenCV and MediaPipe.

## Features

- Real-time hand gesture detection
- Mathematical operations using fingers
- Webcam-based control
- AI hand tracking

## Libraries Used

- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI

## Run Project

```bash
python main.py
```

## Hand Gesture Operations

| Gesture | Operation |
|----------|-----------|
| Index + Thumb | + Addition |
| 2 Fingers + Thumb | - Subtraction |
| 3 Fingers + Thumb | * Multiplication |
| 4 Fingers + Thumb | / Division |
| 1 Finger and 4 Fingers | = Equal |

## How It Works

1. Webcam captures hand.
2. MediaPipe detects fingers.
3. Finger combinations are matched.
4. Calculator operation is performed.
